# admirror brand assets

These files match `app/components/AdmirrorLogo.tsx` (the logo used in the landing and dashboard).

| File | Use |
|------|-----|
| `admirror-logo-on-light.svg` | **Dashboard**, light UI backgrounds (`theme="dark"`) |
| `admirror-logo-on-dark.svg` | **Landing**, login, dark hero (`theme="light"`) |
| `admirror-icon.svg` | Icon only (32×32), dashboard colors |

In the app, open directly:

- http://localhost:3001/brand/admirror-logo-on-light.svg
- http://localhost:3001/brand/admirror-logo-on-dark.svg
- http://localhost:3001/brand/admirror-icon.svg

Source of truth for colors and shapes: `app/components/AdmirrorLogo.tsx`.
